### 文件结构
LED闪烁以及基于IMU的姿态解算的三个任务函数均位于```./Core/Tasks ```文件夹内, 在```./Core/Src/freertos.c ```内被调用.

### 关于电控培训
个人认为电控培训内容太少. 具体表现为**细节指导不足**; 另外**资源紧张**, 没有得到共享. 
我理解团队出于筛选需要, 注重信息素养的考察. 但**作业难度曲线不合理**会使学员产生困扰. 

根据我参加其它机器人社团的经历, 我建议: 
- 限制报名人数, 报满即止, 增加培训课时和内容;
- 将FreeRTOS教程以及校内赛调试提前到整车调试前, 或者减少整车调试考察内容并延长时间. 

电控组的培训确实本身就可以作为一门充实的选修课来看, 所以还是很有意义的.
